# ⏱ Трекер Времени / Time Tracker

Современный однофайловый трекер времени на Vanilla JS. Работает прямо из браузера — без сервера, без зависимостей, без установки.

> **English summary:** A single-file, zero-dependency time tracker. Open `index.html` in any browser — no server needed.

---

## Файлы

| Файл | Описание |
|------|----------|
| `index.html` | Основной файл приложения |
| `demo.html` | Демо-версия с предзаполненными данными (таймеры, история, заметки, достижения) |

## Возможности

- **Несколько таймеров** — создавайте сколько угодно, только один активен в каждый момент
- **Категории** — Работа, Учёба, Личное, Другое; добавление и удаление через диалог ⚙ Категории
- **Режим Помодоро** — кольцо обратного отсчёта, автосмена фаз, звуковые уведомления (работают и при неактивной вкладке — через Web Worker)
- **Mini-виджет (PiP)** — кнопка 📌 открывает плавающее окно поверх всех приложений с текущим таймером и кнопкой ▶/⏸
- **Тёмная / светлая тема** — без мерцания при загрузке (FOUC prevention)
- **Язык интерфейса** — русский / английский, кнопка с флагом в шапке
- **Размер шрифта** — настройка от 12 до 24 px прямо в интерфейсе
- **Мобильная поддержка** — адаптивная сетка, кнопка удаления всегда видна на touch-устройствах
- **Записная книжка** — текстовое поле для заметок, сохраняется автоматически
- **Статистика** — сводка по таймерам, столбчатая диаграмма за 7 дней
- **Экспорт** — CSV (с BOM для корректного Excel) и JSON
- **Лог активности** — последние 200 событий
- **Геймификация** — серия дней подряд 🔥 с рекордом, 12 достижений (время, streak, Помодоро, экспорт, заметки), тост-уведомление при разблокировке
- **Первый запуск** — при пустом хранилище автоматически создаются стартовые таймеры и открывается инструкция
- **Сохранение при перезагрузке** — активный таймер продолжает отсчёт после закрытия вкладки
- **Горячие клавиши** — полное управление с клавиатуры

## Быстрый старт

1. Скачайте `index.html`
2. Откройте в браузере (Chrome, Firefox, Edge, Safari)
3. При первом открытии появятся стартовые таймеры и инструкция

## Mini-виджет (Picture-in-Picture)

Кнопка **📌** в тулбаре открывает компактное окно, которое плавает поверх всех приложений — IDE, мессенджеры, документы.

Виджет показывает название таймера, текущее время (или обратный отсчёт Помодоро), категорию / фазу и кнопку ▶/⏸. Повторный клик на 📌 закрывает виджет.

**Поддержка по платформам:**

| Платформа | Режим | Требования |
|-----------|-------|------------|
| Chrome / Edge (десктоп) | Document PiP — произвольный HTML | Chrome / Edge 116+ |
| Android Chrome | Video PiP — canvas-виджет | Chrome Android 105+ |
| iOS / Firefox / Safari | ❌ недоступно | — |

На десктопе виджет — полноценное окно с кастомным интерфейсом. На Android — видеопоток с canvas-рендером таймера; кнопки ▶/⏸ системного оверлея синхронизированы с состоянием таймера.

## Горячие клавиши

| Клавиша     | Действие                         |
|-------------|----------------------------------|
| `N`         | Добавить таймер                  |
| `Space`     | Пауза / запуск активного таймера |
| `T`         | Переключить тему                 |
| `+` / `-`   | Увеличить / уменьшить шрифт      |
| `1` – `9`   | Активировать таймер по позиции   |
| `Delete`    | Удалить активный таймер          |

## Схема хранения (localStorage)

| Ключ              | Содержимое                                          |
|-------------------|-----------------------------------------------------|
| `timers`          | Массив таймеров с состоянием и Помодоро-данными     |
| `preferences`     | Тема, размер шрифта, язык, настройки Помодоро, звук |
| `notebookContent` | Текст записной книжки                               |
| `categories`      | Категории с цветами                                 |
| `timerHistory`    | История удалённых таймеров (90 дней)                |
| `activityLog`     | Лог действий (200 записей)                          |
| `streaks`         | Серии дней и достижения                             |
| `schemaVersion`   | Версия схемы для автомиграции                       |

## Технологии

- **Vanilla JS** — без фреймворков и зависимостей
- **CSS Custom Properties** — полная темизация через переменные
- **Document Picture-in-Picture API** — плавающий мини-виджет на десктопе (Chrome/Edge 116+)
- **Video Picture-in-Picture API + Canvas captureStream** — мобильный PiP-виджет (Android Chrome)
- **Web Audio API** — звуки без внешних файлов; автовозобновление AudioContext при возврате на вкладку
- **Web Worker** — фоновый таймер для Помодоро: срабатывает даже при неактивной вкладке
- **localStorage** — персистентность данных
- **`<dialog>`** — нативные модальные окна
- **requestAnimationFrame** — обновление только при активном таймере
- **SVG data URI** — favicon и флаги языков без внешних файлов

# ⏱ Трекер Времени / Time Tracker

Современный однофайловый трекер времени на Vanilla JS. Два режима — классический 2D и иммерсивный 3D. Работает прямо из браузера без сервера, без зависимостей, без установки.

> **English summary:** A single-file time tracker in two modes — classic 2D (`index.html`) and immersive 3D (`chronos.html`). Zero dependencies for 2D; Three.js CDN for 3D. Open in any browser, no server needed.

---

## Файлы

| Файл | Описание |
|------|----------|
| `index.html` | Классический 2D-трекер — полный функционал в привычном интерфейсе |
| `chronos.html` | 3D-версия «Хронос» — таймеры как светящиеся кристаллы в трёхмерной сцене |

Между режимами можно переключаться кнопками **✦ 3D** / **⬛ 2D** в шапке.

## Возможности

### Общие для обеих версий

- **Несколько таймеров** — создавайте сколько угодно, только один активен в каждый момент
- **Категории** — Работа, Учёба, Личное, Другое; добавление и удаление через диалог
- **Режим Помодоро** — обратный отсчёт, автосмена фаз (работа / перерыв / длинный перерыв), звуковые уведомления
- **Фоновый Web Worker** — Помодоро срабатывает даже при неактивной вкладке
- **Mini-виджет (PiP)** — кнопка 📌 открывает окно поверх всех приложений с таймером и кнопкой ▶/⏸
- **Тёмная / светлая тема** — без мерцания при загрузке (FOUC prevention)
- **Язык** — русский / английский, кнопка с SVG-флагом в шапке
- **Записная книжка** — сохраняется автоматически
- **Статистика** — сводка и столбчатая диаграмма за последние 7 дней
- **Экспорт** — CSV (с BOM для корректного Excel) и JSON
- **Лог активности** — последние 200 событий
- **Геймификация** — серия дней подряд 🔥, 12 достижений (время, streak, Помодоро, экспорт, заметки)
- **Сохранение состояния** — активный таймер продолжает отсчёт после закрытия и перезагрузки вкладки
- **Горячие клавиши** — полное управление с клавиатуры

### Только в 2D (`index.html`)

- Адаптивная сетка карточек, оптимизированная для мобильных устройств
- Рандомайзер — случайное число или выбор из списка
- Настройка размера шрифта (12–24 px) прямо в интерфейсе

### Только в 3D (`chronos.html`)

- Каждый таймер — светящийся **кристалл** (икосаэдр) в трёхмерной сцене на Three.js
- Вращение / зум / сдвиг мышью; клик по кристаллу — мгновенный старт/пауза таймера
- Кольцо Помодоро вокруг кристалла с анимацией прогресса
- 3D **столбчатая диаграмма** активности за 7 дней прямо в сцене
- 3D **медали достижений** (октаэдры); клик по медали — попап с описанием и статусом
- Кнопки **Таймеры / Статистика / Достижения** — плавный полёт камеры к нужной области

## Быстрый старт

```
# Просто откройте файл в браузере:
index.html    — 2D-версия
chronos.html  — 3D-версия (требует Chrome / Edge для всех возможностей)
```

При первом открытии автоматически создаются стартовые таймеры и открывается инструкция.

## Mini-виджет (Picture-in-Picture)

Кнопка **📌** открывает компактное окно, которое плавает поверх всех приложений — IDE, мессенджеры, документы. Показывает название таймера, время (или обратный отсчёт Помодоро), категорию / фазу и кнопку ▶/⏸.

| Платформа | Режим | Требования |
|-----------|-------|------------|
| Chrome / Edge (десктоп) | Document PiP — произвольный HTML-интерфейс | Chrome / Edge 116+ |
| Android Chrome | Video PiP — canvas-рендер таймера | Chrome Android 105+ |
| iOS / Firefox / Safari | ❌ недоступно | — |

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `N` | Добавить таймер |
| `Space` | Пауза / запуск выбранного таймера |
| `T` | Переключить тему |
| `+` / `-` | Увеличить / уменьшить шрифт |
| `1` – `9` | Выбрать таймер по позиции |
| `Delete` | Удалить выбранный таймер |

## Хранилище (localStorage)

Оба файла используют **одно и то же хранилище** — таймеры, созданные в 2D, сразу видны в 3D и наоборот.

| Ключ | Содержимое |
|------|------------|
| `timers` | Таймеры с состоянием и Помодоро-данными |
| `preferences` | Тема, шрифт, язык, настройки Помодоро, звук |
| `notebookContent` | Текст записной книжки |
| `categories` | Категории с цветами |
| `timerHistory` | История удалённых таймеров |
| `activityLog` | Лог действий (200 записей) |
| `streaks` | Серии дней и достижения |
| `_chronos_meta` | 3D-специфичные данные: дневная активность, счётчик Помодоро |

`index.html` — эталонный формат. `chronos.html` читает и пишет те же ключи через адаптерный слой; при первом открытии chronos после старой версии данные мигрируются автоматически.

## Технологии

### 2D

- **Vanilla JS** — без фреймворков и зависимостей
- **CSS Custom Properties** — полная темизация через переменные
- **Web Audio API** — звуки без внешних файлов
- **Web Worker** — фоновый таймер Помодоро
- **Document / Video Picture-in-Picture API** — плавающий виджет
- **`<dialog>`** — нативные модальные окна
- **SVG data URI** — favicon и флаги языков

### 3D (дополнительно)

- **Three.js 0.160** — WebGL-рендер, физически корректные материалы (MeshPhysicalMaterial)
- **OrbitControls** — управление камерой мышью / тачем
- **CSS2DRenderer** — HTML-метки (название, время, категория) поверх 3D-сцены
- **ACESFilmic tone mapping** — кинематографический рендер

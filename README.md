# ⏱ Трекер Времени / Time Tracker

Современный однофайловый трекер времени на Vanilla JS. Работает прямо из браузера — без сервера, без зависимостей, без установки.

> **English summary:** A single-file, zero-dependency time tracker. Open `index.html` in any browser — no server needed.

---

## Возможности

- **Несколько таймеров** — создавайте сколько угодно, только один активен в каждый момент
- **Категории** — Работа, Учёба, Личное, Другое; добавление и удаление через диалог ⚙ Категории
- **Режим Помодоро** — кольцо обратного отсчёта, автосмена фаз, звуковые уведомления (работают и при неактивной вкладке — через Web Worker)
- **Mini-виджет (PiP)** — кнопка 📌 открывает плавающее окно поверх всех приложений с текущим таймером и кнопкой ▶/⏸; требует Chrome / Edge 116+
- **Тёмная / светлая тема** — кнопка показывает текущее состояние (☀️ = сейчас светлая, 🌙 = сейчас тёмная), без мерцания при загрузке
- **Язык интерфейса** — русский / английский, кнопка с флагом и кодом (RU / EN) в шапке
- **Размер шрифта** — настройка от 12 до 24 px прямо в интерфейсе
- **Мобильная поддержка** — адаптивная сетка, кнопка удаления всегда видна, свайп влево для удаления
- **Записная книжка** — текстовое поле для заметок, сохраняется автоматически
- **Статистика** — сводка по таймерам, столбчатая диаграмма за 7 дней
- **Экспорт** — CSV (с BOM для корректного Excel) и JSON
- **Лог активности** — последние 200 событий (автоочистка, не влияет на производительность)
- **Геймификация** — серия дней подряд 🔥 с рекордом, 12 достижений (время, streak, Помодоро, экспорт, заметки), тост-уведомление при разблокировке; скрываемая панель
- **Сохранение при перезагрузке** — активный таймер продолжает отсчёт после закрытия вкладки
- **Горячие клавиши** — полное управление с клавиатуры

## Быстрый старт

1. Скачайте `index.html`
2. Откройте в браузере (Chrome, Firefox, Edge, Safari)
3. Нажмите **+ Добавить** и создайте первый таймер

---

## Mini-виджет (Picture-in-Picture)

Кнопка **📌** в тулбаре открывает компактное окно (210×160 px), которое плавает поверх всех приложений — IDE, мессенджеры, документы.

Виджет показывает:
- Название активного таймера
- Текущее время (или обратный отсчёт Помодоро)
- Категорию / фазу
- Кнопку ▶ / ⏸

Повторный клик на 📌 закрывает виджет. Требует **Chrome или Edge 116+** (Document Picture-in-Picture API). В Firefox/Safari кнопка недоступна.

## Горячие клавиши

| Клавиша     | Действие                         |
|-------------|----------------------------------|
| `N`         | Добавить таймер                  |
| `Space`     | Пауза / запуск активного таймера |
| `T`         | Переключить тему                 |
| `+` / `-`   | Увеличить / уменьшить шрифт      |
| `1` – `9`   | Активировать таймер по позиции   |
| `Delete`    | Удалить активный таймер          |

## Схема хранения (localStorage)

| Ключ              | Содержимое                                          |
|-------------------|-----------------------------------------------------|
| `timers`          | Массив таймеров с состоянием и Помодоро-данными     |
| `preferences`     | Тема, размер шрифта, язык, настройки Помодоро, звук |
| `notebookContent` | Текст записной книжки                               |
| `categories`      | Категории с цветами                                 |
| `timerHistory`    | История удалённых таймеров (90 дней)                |
| `activityLog`     | Лог действий (200 записей)                          |
| `streaks`         | Серии дней и достижения                             |
| `schemaVersion`   | Версия схемы для автомиграции                       |

## Технологии

- **Vanilla JS** — без фреймворков и зависимостей (jQuery удалён)
- **CSS Custom Properties** — полная темизация через переменные
- **Document Picture-in-Picture API** — плавающий мини-виджет поверх всех окон (Chrome/Edge 116+)
- **Web Audio API** — звуки без внешних файлов; автовозобновление AudioContext при возврате на вкладку
- **Web Worker** — фоновый таймер для Помодоро: звук срабатывает даже при неактивной вкладке
- **localStorage** — персистентность данных
- **`<dialog>`** — нативные модальные окна (вместо `prompt`/`confirm`)
- **requestAnimationFrame** — обновление только при активном таймере
- **SVG data URI** — флаги языков без внешних файлов
